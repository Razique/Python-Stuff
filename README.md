### How to run the game? ###
![Armory](https://raw.github.com/Razique/Python-Stuff/master/gothonweb/static/armory.jpg)  

1. Run the following command (as root):  
`pip install lpthw.web nose`

2. `cd gothonweb`  
3. `python bin/app.py`  
4. Open your browser, and go to:  
http://localhost:8080
